
1. Go to your repository **Settings**

3. Under **Source**, select **main** branch

## â“ Troubleshooting

**Page not loading?**
- Wait 5 minutes after enabling Pages
- Check GitHub Pages settings (must have main branch selected)
- Clear browser cache (Ctrl+Shift+Delete)

**File not showing?**
- Ensure it's renamed to `index.html`
- Check file is in the root of repository
- Verify repository is Public

**Downloads not working?**
- This is a CORS issue with GitHub Pages
- Download links work fine locally
- Use PDF download for best results
